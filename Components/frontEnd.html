FRONT-END

El "front-end" o "frontend" se refiere a la parte de un sitio web, aplicación o software que los usuarios ven y con la que interactúan directamente. Es la cara visible de un producto digital y abarca todo lo que se muestra en el navegador o la interfaz del usuario. El front-end es la interfaz que permite a los usuarios realizar acciones, navegar por el contenido, enviar datos y recibir respuestas.
El front-end se construye principalmente utilizando tecnologías web como HTML (HyperText Markup Language) para estructurar el contenido, CSS (Cascading Style Sheets) para definir el diseño y el aspecto visual, y JavaScript para agregar interactividad y funcionalidad dinámica.
